This folder contains all graphical assets, including logos and backend images, used in the development of the Propylia application. These assets support the graphical user interface (GUI), enhance usability, and ensure a consistent visual identity across all modules of the tool. The images are utilized within feature panels, result visualization sections, and background components corresponding to the following functionalities:

Key features supported by these assets include:

Amino Acid Composition (AAC)

Dipeptide Composition

Tripeptide Composition

Pseudo Amino Acid Composition (PAAC)

Amphiphilic Pseudo Amino Acid Composition (APAAC)

Composition, Transition, and Distribution (CTD)

Autocorrelation Descriptors (Moran)

Result Visualization and Summary Panels

File Management and Help Interface
